# Vehicle CO2 Emissions Analysis

This project analyzes a 2023 Canadian vehicle dataset to study the relationship between vehicle characteristics, fuel consumption, and CO2 emissions.

## Contents
- `data/` dataset used for analysis
- `notebooks/` Jupyter notebook containing EDA and modeling
- `reports/` supporting project report

## Methods
- Exploratory data analysis
- Linear regression
- Ridge regression
- Classification tree
- Random forest

## Goal
Build predictive models for vehicle CO2 emissions and identify the strongest drivers of emissions.
